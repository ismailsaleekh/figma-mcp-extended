About Figma MCP Extended

This plugin allows AI assistants to communicate with Figma, enabling AI-assisted design operations. created by Sonny

Version: 1.0.0

How to Use

  1. Make sure the MCP server is running
  2. Connect to the server using the port number (default: 3055)
  3. Once connected, you can interact with Figma through AI tools
